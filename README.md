# Looker Ecommerce 데이터분석

Looker Ecommerce 데이터셋(Kaggle 제공)을 활용한 데이터분석

---

## 프로젝트 개요

Kaggle의 Looker Ecommerce BigQuery Dataset을 활용한 e-commerce 데이터 분석 프로젝트입니다.
EDA, 유저 군집화, 가설 검정을 통해 비즈니스 인사이트를 도출하였습니다.
---

## 분석 내용

## 분석 내용
## 데이터 탐색 (EDA)

- 카테고리/브랜드/성별/국가별 매출 분석
- 기간별 매출 추이 및 환불 패턴 분석
- Traffic source별 구매 이벤트 탐색

## 심화 분석

- 퍼널 분석: 상품 진입 → 장바구니 → 구매 전환율 분석
- 리텐션/코호트 분석: 주간 리텐션, 가입월별 코호트, 국가별 비교
- K-Means 클러스터링: 유저 4개 군집 분류 (Elbow Method, Silhouette 분석)
- 가설 검정: t-검정, ANOVA, 카이제곱 검정
